# Pilah - Vercel Hosting

Folder ini siap di-upload ke Vercel sebagai static site.

## Isi folder

- `/` dan `/bundle`: halaman akses pembeli.
- `/vol1`: AI Prompt Mastery Vol.1 LOCKED.
- `/vol2`: AI Prompt Mastery Vol.2 LOCKED.
- `/bonus-9-skill`: bonus 9 skill AI LOCKED.
- `/pilah-01`, `/pilah-02`, `/pilah-03`: versi Pilah per volume.

File internal yang TIDAK dimasukkan:

- `GENERATOR-KEY-Pilah.html`
- `keys_master.json`
- file HTML asli tanpa LOCKED

## Cara upload ke Vercel paling mudah

1. Buka Vercel dan login.
2. Buat project baru.
3. Upload folder ini sebagai project static.
4. Framework preset: Other.
5. Build command: kosongkan.
6. Output directory: kosongkan atau gunakan root.
7. Deploy.

Setelah deploy, link pembeli menjadi:

- `https://nama-project.vercel.app/bundle`
- `https://nama-project.vercel.app/vol1`
- `https://nama-project.vercel.app/vol2`
- `https://nama-project.vercel.app/bonus-9-skill`

## Pesan delivery pembeli

```text
Halo, terima kasih sudah order AI Prompt Mastery Bundle.

Ini akses produk kamu:
[LINK VERCEL /bundle]

Access Key:
[KEY PEMBELI]

Cara buka dari HP:
1. Klik link akses dari Chrome/Safari.
2. Pilih produk yang ingin dibuka.
3. Masukkan Access Key.
4. Setelah terbuka, bookmark halaman agar gampang dibuka lagi.

Kalau muncul error atau key tidak valid, kirim screenshot ke saya lewat WA ini.
```

## Catatan penting

- Jangan upload generator key ke Vercel.
- Jangan upload `keys_master.json` ke Vercel.
- Catat key yang dikirim ke pembeli di spreadsheet.
- Test link dari HP sebelum mulai jualan atau iklan.
